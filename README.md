# CyberSecurity Awareness Bot

A Windows Forms Cybersecurity Awareness Chatbot developed in C#.

features
- GUI chatbot
- Voice greeting support
- Keyword recognition
- Sentiment detection
- Memory recall
- Random cybersecurity tips
- Input validation
- Object-Oriented Programming structure


1. Open the solution in Visual Studio
2. Build the project
3. Run the application
